O5. Cách set location trong Facebook Ads Manager

Option B — Pin radius từng salon (chính xác, dùng cho C2):

  1. Add Locations → Drop Pin → nhập địa chỉ salon
  2. Set radius: 3km (quận nhỏ nội đô) · 5km (quận rộng / ngoại ô)
  3. Thêm pin cho từng salon trong cùng ad set → circles chồng lên nhau = phủ hết quận
